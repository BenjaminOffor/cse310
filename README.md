## Project Overview
This is a simple JavaScript quiz program that asks users multiple-choice questions and calculates their final score.  
It runs in a web browser and demonstrates understanding of JavaScript basics: variables, conditional statements, loops, and functions.

## Features
- Multiple-choice questions
- Immediate feedback for correct/incorrect answers
- Final score calculation
- Restart quiz functionality

## How to Run
1. Open `index.html` in any modern web browser.
2. Answer the questions by clicking the answer buttons.
3. Click **Next Question** to proceed.
4. At the end, view your score.
5. Click **Restart Quiz** to try again.

## Requirements
- Runs in a browser (no server required)
- Uses HTML, CSS, and JavaScript
